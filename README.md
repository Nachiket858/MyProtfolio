# Nachiket Shinde - Portfolio

A high-performance Multi-Page Application (MPA) built with **Next.js**, **React**, and **TypeScript**. Designed using a custom "Neural Obsidian" visual aesthetic to showcase expertise in AI systems, workflow automation, and backend engineering.

## 🚀 Tech Stack

- **Core Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Custom CSS (Glassmorphism + "Neural Obsidian" Tokens)
- **Icons:** Bootstrap Icons
- **API Integration:** Web3Forms for secure contact handling
- **Deployment:** Vercel

## ✨ Key Features

- **Blazing Fast MPA Routing:** Utilizes Next.js Server Components and statically generated routes for instant page loads.
- **Search Engine Supreme:** Equipped with dynamically injected Advanced Metadata, Open Graph configuration (for gorgeous URL sharing on LinkedIn/Twitter), a programmatic `sitemap.xml`, and Google Rich Snippet JSON-LD.
- **Persistent VFX Architecture:** A globally scoped, non-blocking Canvas Particle network sits beautifully underneath the routing layer to prevent jitter during page navigation.
- **Intelligent CSS Triggers:** Custom `ScrollReveal` intersections orchestrate entrance animations cleanly and efficiently.

## 🛠️ Local Development & Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Setup environment keys (needed for `/contact` form functioning):**
   Create a `.env.local` file at the root:
   ```env
   NEXT_PUBLIC_WEB3FORMS_ACCESS_KEY="your_web3forms_key_here"
   ```
3. **Launch the development server:**
   ```bash
   npm run dev
   ```
4. **View locally:**
   Open `http://localhost:3000` to interact with the build.
